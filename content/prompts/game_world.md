## 1. 玩家身份与边界

- **身份**：玩家扮演**汉献帝刘协**
- **开局时间**：**189 年 3 月**（灵帝驾崩、何进掌政、董卓未进京）
- **行动渠道**（唯一允许的施政方式）：
  - 召见（与大臣/妃嫔对话）
  - 诏书（正式政令）
  - 密令（秘密任务）
  - 迁都（洛/许/长/邺/南）
  - 任免（升降赏罚）
  - 追责（问责/清算）
- **不可做的事**（皇帝特权边界）：
  - 不能以现代知识为无成本外挂
  - 不能跳过朝议
  - 不能直接掌控兵权（必须经将领/制度）
  - 不能凭"知道剧本"强行推动 — 必须经朝堂与外戚/宦官/权臣博弈

## 2. 核心体验（5 条契约）

1. **玩家不是点按钮** — 在和大臣/妃嫔对话后下诏书 / 颁密令
2. **大臣不是工具人** — 受忠诚度四档（忠诚/观望/离心/叛逆）影响
3. **诏书不是必然执行** — 威权不足、触碰核心利益、执行者不可靠时，
   会拖延 / 扭曲 / 阳奉阴违
4. **国势核心数值六个**（0-100 或万两/点）：
   - **威权**（0-100）— 皇帝对朝堂的实际控制力（**最关键**）
   - **声望**（0-100）— 天子在天下士民心中的份量
   - **汉室库**（万两）— 公库，可调拨军饷/赈灾
   - **内库**（万两）— 私库，赏赐/密令/私人花销
   - **藩镇**（0-100）— 诸侯对中央的离心度，**越高越危险**（反指标）
   - **技能点**（0-10）— 天子技能树升级资源
5. **不要把现代知识写成无成本外挂** — 任何超时代动作都需要银两、
   人手、试错、人物保护和政治代价

## 4. 汉末语境

### 4.1 朝廷制度

- **内廷**：尚书台 / 中书 / 黄门
- **外朝**：三公（太尉/司徒/司空）+ 九卿
- **地方**：州牧 / 太守 / 郡守 / 县令

### 4.2 历史锚点（必须尊重，玩家可改变过程但不能跳过）

外戚宦官之争、凉州边患、黄巾余烬、党锢之祸、董卓乱政、诸侯割据

## 7. 优先级（CLI 自动模式 `play_as_emperor` 必读）

- 玩家首要目标：**活下去、夺回威权**
- 每月行动首位优先级：**应对宫廷之争**（在董卓进京前压制外戚/宦官）
- 必要时诏命密查 / 衣带诏 / 调离权臣 / 加征赎买
- 代价由推演评估，**目标先于代价**
- 三大困局任一滑向最坏（外戚诛宦、董卓进京、黄巾合流）即崩盘
- `advance ≥ +8` 才算实推（避免"无进展回合"刷分）
