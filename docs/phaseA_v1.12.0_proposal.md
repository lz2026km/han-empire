> 以下 6 章节 100-130 行，**所有数据均以 db 实测为准**（不是凭印象）。

```markdown
# 《汉献帝之末路》· 游戏世界观（汉献帝宪法）

## 0. 本文件性质

本文件是本游戏玩法的**唯一权威源**。
`simulator.md` / `season_simulator.md` / `extractor.md` / `memory_extractor.md` /
`minister_agent.md` / `decree_writer.md` / `opening_gazette.md`
中所有玩法规则若与本文件冲突，**以本文件为准**。

---

## 1. 玩家身份与边界

- 玩家扮演**汉献帝刘协**，开局时间 **189 年 3 月**（灵帝驾崩、何进掌政、董卓未进京）
- 玩家知道汉室将衰、知道三国归晋，但**不能直接绕过制度**；皇帝只能通过
  **召见、诏书、密令、迁都、任免、追责** 推动事情
- 不可做的事：
  - 不能以现代知识为无成本外挂
  - 不能跳过朝议、不能直接掌控兵权
  - 不能凭"知道剧本"强行推动 — 必须经朝堂与外戚/宦官/权臣博弈

---

## 2. 核心体验（5 条契约）

1. **玩家不是点按钮** — 是在和大臣/妃嫔对话后下诏书 / 颁密令
2. **大臣不是工具人** — 受忠诚度四档（忠诚/观望/离心/叛逆）影响
3. **诏书不是必然执行** — 威权不足、触碰核心利益、执行者不可靠时，
   会拖延/扭曲/阳奉阴违
4. **国势核心数值六个**（0-100 或万两/点）：
   - **威权**（0-100）— 皇帝对朝堂的实际控制力
   - **声望**（0-100）— 天子在天下士民心中的份量
   - **汉室库**（万两）— 公库，可调拨军饷赈灾
   - **内库**（万两）— 私库，赏赐/密令/私人花销
   - **藩镇**（0-100）— 诸侯对中央的离心度，**越高越危险**
   - **技能点**（0-10）— 天子技能树升级资源
5. **不要把现代知识写成无成本外挂** — 任何超时代动作都需要银两、人手、
   试错、人物保护和政治代价

## 3. 数据契约（实测数据，非凭印象）

### 3.1 核心盘面（6 个量表）

| 字段 | 类型 | 范围 | 说明 |
|------|------|------|------|
| `treasury` / `汉室库` | int | 万两 | 公库，月底由推演结算 |
| `inner_treasury` / `内库` | int | 万两 | 私库 |
| `reputation` / `声望` | int | 0-100 | 士林民间清议 |
| `authority` / `威权` | int | 0-100 | 朝廷控制力（核心） |
| `fanzhen` / `藩镇` | int | 0-100 | 诸侯离心度（**反指标**） |
| `skill_points` / `技能点` | int | 0-10 | 天子技能树 |

### 3.2 派系（两层，勿混用）

**A. 朝廷百官派系（5 派系，影响朝堂博弈）**

| 派系 | 立场 | 特征 |
|------|------|------|
| 忠汉派 | 拥汉 | 重节义，怕董卓恨奸佞 |
| 务实派 | 实用 | 重实效，看皇帝能不能办事 |
| 离心派 | 观望 | 骑墙，看哪边赢就跟哪边 |
| 叛逆派 | 反汉 | 暗中图谋不轨或已起兵 |
| 董卓军 | 凉州 | 残暴、兵强、不受朝堂号令 |

**B. 势力阵营派系（28 种，影响地区/军队控制）**
实测自 `content/characters.json` 人物的 `faction` 字段：
- 曹魏 34人 / 汉室 30人 / 东吴 25人 / 蜀汉 21人 /
  凉州集团 4人 / 袁绍 4人 / 凉州军 3人 / 扬州 3人 /
  刘焉 3人 / 袁氏 2人 / 公孙瓒 2人 / 董卓 2人 / ... 等
- 决定该人物归属的 power_id（汉室/曹魏/蜀汉/东吴/凉州/...）

**两套派系不要混用** — 朝廷百官派系是朝堂博弈层，势力阵营派系是地缘控制层。

### 3.3 阶级（8 阶级，实测自 classes.json）

| 阶级 | 人口(万) | 满意度 | 影响力 | 议程 |
|------|---------|--------|--------|------|
| 流民 | 300 | 15 | 60 | 求活命、就食于城 |
| 羌胡 | 200 | 30 | 45 | 求自治、抗边将暴敛 |
| 寒门 | 120 | 45 | 50 | 求仕进、抗豪族兼并 |
| 商贾 | 80 | 50 | 55 | 通有无、抗重税 |
| 豪族 | 50 | 55 | 80 | 护田产、避赋税、控辟举 |
| 士人 | 30 | 48 | 70 | 清议讽议、议朝政得失 |
| 宗室 | 20 | 55 | 65 | 保宗禄、护藩田、抗削藩 |
| 宦官 | 5 | 60 | 90 | 保权势、干预朝政 |

## 5. 开局三困局（189.3 献帝即面对）

> 这三件事在开局邸报（`opening_gazette.md`）中已铺开，作为 active issues 持续推演。

1. **宫廷之争**：何进与张让赵忠争权；何进纳袁绍策召外兵入京，

## 7. 优先级（CLI 自动模式 `play_as_emperor` 必读）

- 玩家首要目标：**活下去、夺回威权**
- 每月行动首位优先级：**应对宫廷之争**（在董卓进京前压制外戚/宦官）
- 必要时诏命密查 / 衣带诏 / 调离权臣 / 加征赎买
- 代价由推演评估，**目标先于代价**
- 三大困局任一滑向最坏（外戚诛宦、董卓进京、黄巾合流）即崩盘
- `advance ≥ +8` 才算实推（避免"无进展回合"刷分）
```

## 5. 风险与回退

| 风险 | 概率 | 缓解 |
|------|------|------|
| LLM 误解"📜 权威源"标记 | 低 | 标记只是文档说明，LLM 仍按各 prompt 内容工作 |
| 数据字段命名混乱 | 中 | §3 字段全用 `中英文对照表`，避免歧义 |
| 影响现有推演输出 | 低 | **不删任何已有规则**，只新增"权威源声明" |
| game_world 写错 | 中 | 实施后实测"4 派系→5 阶级"等 db 值核对（参考 v1.10.0 教训） |

**回退方案**：`git revert` 该 commit 即可，game_world 是新增文件不破坏其他模块。
