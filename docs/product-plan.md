# 汉献帝之末路 · 产品设计

## 核心定位

**类型**：历史角色扮演沙盘模拟
**玩家**：扮演汉献帝刘协，在董卓乱政、诸侯割据的时代中艰难求存、力图中兴
**目标**：通过诏令、召对、用人、决策，延缓汉室灭亡乃至实现中兴

## 核心玩法循环

```
回合（月）= 召对 + 诏令 → LLM 模拟 → 结果落库 + 记忆生成 → 回合报告
```

1. **召对**：与大臣对话，了解局势、获取建议、观察态度
2. **拟旨**：将口谕转为正式诏书（LLM 润色）
3. **执行**：诏书通过 LLM 推演落地（数值变化 + 叙事生成）
4. **记忆**：系统提取本回合关键事件为记忆，影响后续召对
5. **报告**：生成月末奏章，呈献帝御览

## 核心数值

| 数值 | 说明 | 变化来源 |
|------|------|----------|
| 威权 | 天子权威，影响技能解锁 | 都城、事件、战绩 |
| 汉室库 | 中央财政收入 | 田赋、盐铁、关税 |
| 内库 | 天子可自由支配 | 查抄、赏赐、特权收入 |
| 民心 | 天下黎民态度 | 赋税、赈灾、军事胜败 |

## 特色机制

- **威权系统**：随都城迁移和重大事件剧烈波动
- **迁都机制**：洛阳→长安→许昌三都流转，每迁都有代价
- **历史锚点**：25个关键时间节点必须尊重（董卓进京/诸侯讨董/赤壁/禅让等）
- **董卓伏诛**：基于威权/军事/事件综合判定
- **献帝东归**：董卓死后的生存路线选择
- **藩镇动态**：各地节度使/太守自主决策

## 派系系统

- 何进派（外戚）、董卓派（凉州军）、袁绍派（世家大族）
- 清流派（士人）、宦官派（内廷）
- 曹操派（寒门功臣）、刘备派（汉室宗亲）
